# Version

Running `sedge version` will give you the version you are running while using sedge.

## Help

```
$ sedge version -h
Initializing configuration
Print sedge version

Usage:
  sedge version [flags]

Flags:
  -h, --help   help for version

Global Flags:
      --logLevel string   Set Log Level (default "info")
```

## Execution Example

```
$ sedge version
2022-00-00 00:00:00 -- [INFO] [Logger Init] Log level: info
2022-00-00 00:00:00 -- [INFO] You are running the latest version of sedge. Version:  v0.5.1
sedge v0.5.1
```
